CVS Repository Information
 

Adding or Updating a file in the repository

set these environmental variables first, or make sure they are set:

setenv CVS_RSH ssh
setenv CVSROOT csf.cs.utah.edu:/csafe_noexport/cvs/cvsroot
setenv CVSUMASK 000
 

To add a new directory to the repository, go to the directory where you want it to be and,

cvs add <dirname>
 

To add a new file to the repository, in the directory where you want the file to appear,

cvs add <filename>
 

To commit changes to a file to the repository

cvs commit -m "explain your changes" <filename>
 

Access to the repository is password-protected. Your password is the same as your
Unix password.  If you get any errors, first double check your environment settings,
then send mail to scirun-develop@cs.
