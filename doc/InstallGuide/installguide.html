SCIRun Installation Guide

(Unix version - updated April 6, 2001)

I Quick Start SCIRun Installation 

      A quick check list for installing and building
      SCIRun and packages.

II SCIRun Installation and Build Instructions 

      Detailed information and instructions for installing
      and building SCIRun and packages.  Includes sections
      on known issues (FAQ), System Requirements, PC system
      setups that have been tested for use with SCIRun and
      more.

III Thirdparty Software Installation and Build Information 

      Detailed information for automatically or
      manually installing the required and optional
      thirdparty software.  Includes a section on known
      issues with building and installing the required
      thirdparty software.
