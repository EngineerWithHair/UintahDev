1. Introduction

The SCIRun User's Guide describes how to use the SCIRun problem solving environment for users who will build and execute networks within the SCIRun environment.

SCIRun installers should read the SCIRun Installation Guide

   
1.1 Road Map

The User's Guide is organized into the following main sections:

Introduction (this section)
Concepts Introduces an integrated problem solving environment, and describes how SCIRun embodies some of these ideas.
Starting SCIRun Starts SCIRun, and contains related information.
Working with Networks Discusses tasks involved in building, editing, and executing networks.
Visualization with the Viewer Describes the purpose and use of the Viewer, SCIRun's most commonly used visualization module.

   
1.2 Getting Help

Help is available from several sources:

1.2.1 In the Distribution

This document and other related documents may be found in your SCIRun distribution. Point your browser at index.html which is in your distribution's top level doc directory (i.e., <top of distribution>/doc/index.html). Latex and postscript versions of some documents are also in the distribution's doc directory.

1.2.2 On the Web

This and other SCIRun documents can be found in the document section of the SCI web site.

Visit the main SCI web site to access other information about SCIRun and the SCI Institute.

1.2.3 From the Mailing Lists

The SCIRun users mail list is a forum for discussing SCIRun related issues. To subscribe send mail to:

Majordomo@cs.utah.edu

with the following command in the body of your message:

subscribe scirun-users

   
1.3 Reporting Bugs

Please report bugs! To report a bug visit SCIRun's bug database web page.

Reporting bugs through the web site (rather than by way of the mailing list) ensures their timely repair.
