1. Introduction

This is the BioPSE User's Guide. It describes the purpose and use of the BioPSE problem solving environment. It is for those users who will be building and executing networks within the BioPSE environment.

Those who will be installing BioPSE should read the BioPSE Installation Guide.

Those wishing to develop modules should read the BioPSE Developer's Guide.

1.2 Roadmap

This document is organized into the following main sections:

Introduction You are reading this section now.
Concepts Introduces the concept of an integrated problem solving environment generally and describes how BioPSE embodies some of these ideas.
Starting SCIRun Procedure for starting BioPSE and related information.
Working with Networks Discusses tasks involved in building, editing, and executing networks.
Visualization with the Viewer Describes the purpose and use of a visualization module called the Viewer. The Viewer is probably BioPSE's most commonly used module.
Packages Describes the content of the SCIRun and BioPSE packages (which provide nearly all of BioPSE's computational services).
