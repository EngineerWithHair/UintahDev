Globus

Use the make-globus script to build the globus library. The make-globus script will use cvs to checkout the globus (and other necessary codes) tarbal from the dist directory, and then will proceed to compile them.
  • > cd ./SCIRun/src/ThirdParty
  • If you want a 64 bit build, edit ../scripts/make-globus and change CFLAG=-O to CFLAG="-O -64" (on both lines.)
  • > ../scripts . .. -lfetchop
