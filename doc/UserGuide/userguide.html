SCIRun User's Guide

(Unix version - Updated May 2, 2001)

I SCIRun User's Guide

Getting started with SCIRun. Includes discussions of the dataflow programming paradigm and problem solving environments. Explains basic concepts of how to run and use SCIRun, and how to write dataflow programs using SCIRun. Also includes an in depth section on the Viewer module.

II SCIRun Modules

Descriptions of all the SCIRun modules, including datatypes used, functions performed and explanations of the user interface elements.
