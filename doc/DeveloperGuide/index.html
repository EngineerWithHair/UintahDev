SCIRun Developer Guide

UNIX version - Updated July 13, 2001

The Developer's Guide contains information in a how-to format for tasks related to writing SCIRun modules. Topics such as memory managment, thread managment, data input and output and dataflow programming are addressed.

1 3D Widgets and Constraints in SCIRun

This chapter introduces the concepts of 3D widgets and the Constraints used to make them functional.

2 Deferred Compilation

This chapter introduces the concept of Deferred Compilation within SCIRun.

3 The SCIRun Interface to Tcl

This chapter describes how to easily communicate between the GUI (Tcl/Tk) and the C++ elements of SCIRun modules.

4 SCIRun Memory Management

This chapter explains how and why SCIRun manages memory usage.

5 Persistent Data

This chapter explains how SCIRun reads and writes data to and from streams.

6 Custom Tk Widgets for SCIRun

This chapter describes extensions made to Tk by SCIRun to provide for custom widgets.

7 SCIRun Utilities

This chapter describes some simple utilities available in SCIRun.

8 SCIRun Exceptions

This chapter explains SCIRun exceptions.

9 Dataflow Programming

This chapter describes the elements of a SCIRun dataflow program.
